## 🎯 이 차시의 학습 목표

- [ ] `sphere`·`box`·`cylinder` 세 줄로 등장인물 3종을 무대에 올릴 수 있다
- [ ] 같은 객체라도 매개변수에 따라 모양이 달라짐을 안다
- [ ] 이 베이스라인을 내 프로젝트에 가져갈 때 ★ 마커 위치를 모두 안다

## 들어가며 — "3D판 hello world"

코드 첫 학습이 `print('hello world')`라면, VPyLab의 첫 신호는 **공·상자·기둥** 한 개씩입니다. 캐릭터·건물·행성·세포 — 무엇을 만들든 이 셋의 변형으로 출발합니다.

## 핵심 개념 — 도형 3종과 크기 표현

| 도형 | 어울리는 용도 | 크기 인자 |
|---|---|---|
| `sphere` | 행성·머리·입자 | `radius=0.7` (반지름 한 값) |
| `box` | 건물·블록 | `size=vector(...)` (세 변) |
| `cylinder` | 다리·기둥·화살 | `axis=vector(...)` + `radius` |

`axis`는 방향과 길이를 한 번에 말하는 화살표. `vector(1.2, 0, 0)`이면 옆으로 누운 기둥, `vector(0, 0, 1.2)`면 화면 안쪽으로.

## 🧰 베이스라인으로 가져갈 때 바꿀 곳

| ★ 위치 | 무엇을 바꾸나 | 결과 변화 |
|---|---|---|
| `pos=vector(...)` | 무대 위 자리 | 등장인물 위치 |
| `radius` | 공 크기 | 주연/조연 비중 |
| `color=색상[...]` | 색 | 정체성·팀 구분 |
| `box`의 `size` | 변 길이 | 책상↔빌딩 |
| `cylinder`의 `axis` | 방향·길이 | 가로·세로·대각선 |

## ⚠️ 자주 하는 실수

!!! warning "셋이 한 점에 겹쳐 보일 때"
    1. 셋 다 `pos=vector(0, 0, 0)` — x를 -2, 0, 2처럼 띄우기.
    2. `box`의 `size`에 단일 숫자(`1.2`) — 반드시 `vector(...)`.
    3. `cylinder`에 `axis` 누락 → 기본값으로 옆으로 눕습니다.
    4. 어두운 색이 검은 배경에 묻힘 — 밝은 색 우선.
