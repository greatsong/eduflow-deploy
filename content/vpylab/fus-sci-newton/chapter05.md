## 🎯 이 차시의 학습 목표

- [ ] 단진자의 운동방정식 `d²θ/dt² = −(g/L)·sin θ`를 코드 세 줄로 옮길 수 있다
- [ ] 각도, 각속도, 각가속도가 *서로 적분으로 연결된다*는 사실을 안다

## 들어가며 — "흔들림에는 패턴이 있다"

벽시계의 추가 *똑딱똑딱* 일정한 박자로 흔들리는 모습. 갈릴레오가 성당에서 *샹들리에가 흔들리는 주기*를 자기 맥박으로 측정해 발견한 사실이 있죠 — *"진자의 주기는 진폭에 무관하다"* (단, 진폭이 작을 때).

이 차시 코드는 지금까지 중 가장 *수식이 진하게* 들어갑니다. 하지만 핵심은 같습니다. *시간을 dt로 잘게 자르고, 매 칸마다 변화량을 더한다.*

```text
알파 = -(g / L) * math.sin(세타)   # 각가속도
오메가 = 오메가 + 알파 * dt          # 각속도
세타 = 세타 + 오메가 * dt            # 각도
```

세 줄이 곧 진자의 운동방정식입니다.

## 핵심 개념 1 — 각도판 뉴턴 제2법칙

직선 운동에서 *위치 → 속도 → 가속도*가 미분으로 연결되듯, 회전 운동에서는 *각도(θ) → 각속도(ω) → 각가속도(α)*가 미분으로 연결됩니다.

| 직선 | 회전 | 코드 변수 |
|---|---|---|
| 위치 x | 각도 θ | `세타` |
| 속도 v | 각속도 ω | `오메가` |
| 가속도 a | 각가속도 α | `알파` |
| F = ma | τ = Iα | (시뮬에선 직접 안 씀) |

뉴턴 제2법칙을 *진자 끝 점*에 적용하면 다음 식이 나옵니다.

> **d²θ/dt² = −(g/L)·sin θ**

좌변은 각가속도(`알파`), 우변은 *복원력에 의한 가속도*. 마이너스가 붙는 이유는 *복원* 때문입니다. 추가 오른쪽으로 가면(θ > 0) 중력이 추를 *왼쪽으로* 끌어당겨 다시 평형 위치(θ = 0)로 돌리려 하죠. 부호가 *방향*을 표현한다는 점, 자유낙하의 `-9.8`과 같은 원리입니다.

세 줄 코드를 *적분의 관점*으로 다시 보면:

- 첫 줄: 식에서 `α`를 직접 계산
- 둘째 줄: ω = ∫α dt → "각가속도 누적해서 각속도 만들기"
- 셋째 줄: θ = ∫ω dt → "각속도 누적해서 각도 만들기"

1차시에서 배운 *잘게 잘라 더하기*를 두 번 연속으로 적용한 것입니다.

## 핵심 개념 2 — 작은 각 근사: `sin θ ≈ θ`

물리책에는 단진자 주기 공식이 다음처럼 나옵니다.

> **T = 2π √(L/g)**

이 공식이 성립하는 조건이 바로 **작은 각 근사**입니다. θ가 작을 때(약 15° 이내) `sin θ ≈ θ`(라디안)로 근사할 수 있어, 운동방정식이 다음처럼 *단순한 단순조화운동* 식이 됩니다.

> d²θ/dt² ≈ −(g/L)·θ

이게 `sin` 대신 `θ`만 등장하는 식이고, 여기서부터 주기 공식이 깔끔하게 풀립니다. 우리 코드의 시작 각도는 45°인데, 이는 *근사가 약간 깨지는 영역*입니다. 그래서 코드는 `math.sin(세타)`을 *원형 그대로* 사용합니다 — 정확한 운동방정식 그대로 시뮬하는 거죠.

학생들이 `math.sin(세타)`을 `세타`로 바꿔 시뮬해 보면 *큰 각도에서 두 결과가 점점 어긋난다*는 점을 직접 확인할 수 있습니다.

## ⚠️ 자주 하는 실수

!!! warning "진자 시뮬레이션 주의 사항"
    1. **각도 단위 혼동** — `세타 = 45`로 적으면 *45 라디안*(약 8 바퀴!)이 되어 시뮬이 날뜁니다. 반드시 `math.radians(45)`.
    2. **순서 뒤바뀜** — `세타`를 먼저 갱신하면 (반암시적 오일러), 결과가 미세하게 다릅니다. 우리 예제는 *명시적 오일러*.
    3. **`줄.clear()`를 안 함** — `줄`을 매 프레임 초기화하지 않으면 *옛 곡선이 남아 거미줄*이 됩니다.
    4. **L과 g의 비율 무시** — T = 2π√(L/g)이니 L을 4배로 늘리면 주기는 2배. 직접 시뮬해 발견하게.

## 🧑‍🏫 교사 노트

!!! note "수업 운영 팁 — 갈릴레오 따라하기"
    실제 진자 실험은 학생들에게 매우 인상적입니다. 줄과 추로 진자를 만들고, 스마트폰 스톱워치로 *10번 진동 시간*을 재서 1주기 시간을 구합니다. 그 값을 시뮬과 비교하세요.

    - 발견 활동: 줄 길이를 *바꾸면 주기가 변하지만*, 추 무게나 초기 각도(작을 때)를 바꿔도 *주기는 거의 안 변한다*. 이게 갈릴레오의 발견.
    - 코드 챌린지: 시작 각도를 `math.radians(170)` 정도로 *큰 각도*로 두면 진자가 *공중제비*를 돌 수도. (수치적으로는 발산 가능 — 좋은 토론 소재.)
    - 평가 아이디어: "1초 주기 진자의 줄 길이는?" T = 2π√(L/g) = 1 → L ≈ 0.248 m. 이론값 도출 후 시뮬·실측으로 검증.
    - 과학+정보 협업: 정보 교사는 *2계 미분방정식을 두 변수로 분해*하는 기법을, 과학 교사는 *복원력의 의미*를 분담.
